What I can

デザインから、CMS導入まで
多岐に渡り
ご依頼を承っております。

制作するサイトの目的、方向性、デザインなどが明確にな
るよう、円滑にコミュニケーションを取ります。
現在クライアント様が抱えている課題が解決できるよう、
ヒアリングでしっかりと課題を把握していきます。

使用言語・ツール

ヒアリングシート

ワイヤーフレームからデザインカンプまでヒアリングで決
まった、方向性やデザインをもとに制作していきます。丁
寧なヒアリングを心がけ、クライアント様のご要望に答え
られるようなデザインを作ります。

使用言語・ツール

Figma

デザインを参考にHTML＆CSSで、javascriptを用いて
Webサイトを作成いたします。CSS設計を意識し、保守・
運用がしやすいコードを書くことを意識しています。

使用言語・ツール

HTML / CSS / JavaScript / VSCode

HTML&CSS、JavaScriptでコーディンしたサイトを更に
保守・運用がしやすくなるようPHPを用いてwordpress
化します。

使用言語・ツール

wordpress/ PHP
